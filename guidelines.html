Guidelines

The GeoHealth Web Application is a comprehensive tool designed to analyze healthcare accessibility in Kathmandu Valley. Using geospatial data and interactive maps, it empowers users to locate nearby hospitals within a specified radius and visualize spatial coverage with buffer zones. The application leverages modern geospatial technologies to address real-world challenges in health accessibility.

Key Objectives

  • Provide an interactive platform for users to locate hospitals within a specific radius.
  • Visualize healthcare accessibility using geospatial data and buffer zones.
  • Enhance decision-making for health services planning and resource allocation.

Technology Stack

The application is built using a robust technology stack, integrating advanced geospatial tools and frameworks:

  • Frontend: HTML, CSS, JavaScript for interactive maps and user-friendly interface.
  • Backend: GeoDjango framework for managing geospatial data and functionalities.
  • Database: PostgreSQL with PostGIS extension for spatial data storage and querying.
  • APIs: GeoServer for hosting shapefiles and hospital data, Weather API for real-time weather information.

Functionality of Components

  • Interactive Mapping: Users can interact with the map to specify a location and radius in meters.
  • Hospital Locator: The system calculates Euclidean distances to find and display hospitals within the user-defined radius.
  • Buffer Visualization: Buffer zones are dynamically rendered on the map, showcasing spatial coverage.
  • Base and Overlay Layers: Includes OpenStreetMap, Esri Imagery, Carto Light, and Dark Matter as base layers, with overlays for Kathmandu Valley boundaries, hospitals, and weather data.

Step-by-Step User Guide

  • Launch the GeoHealth Web Application:
    • Open the GeoHealth web application in your browser.
    • The map will be displayed as the main interface.
    • The interactive map will allow you to explore Kathmandu Valley and other areas.
  • Locate a Point on the Map:
    • Click anywhere on the map to select a specific location.
    • The clicked location will be marked on the map with a marker.
    • You can adjust the marker's position to precisely select the location of interest.
  • Input a Radius:
    • Find the input field labeled "Enter Radius (meters)".
    • Enter the radius in meters (the distance from the selected point within which you want to search for hospitals).
    • Click the "Submit" button to apply the radius.
  • View Hospitals within the Radius:
    • The application will send a request to the server with the selected location and radius.
    • Hospitals within the specified radius will be retrieved from the server.
    • Markers representing the hospitals will appear on the map, showing their exact locations.
    • You can click on any hospital marker to see more details such as the hospital name, type, and other relevant information.
  • Visualize the Buffer Zone:
    • A circular buffer zone will be drawn on the map, centered at the selected point.
    • This buffer zone visually highlights the area within which the hospitals are located, helping you easily see the search area.
  • Toggle Base Maps:
    • You can switch between different base maps to customize the map view.
    • Available base maps include OpenStreetMap, Esri Imagery, Carto Light, and Dark Matter.
    • Base maps can be toggled via the layer control menu on the map.
  • View Hospital Data and Weather Information:
    • The application will display hospital data (hospital names, types) on the map.
    • The weather data for the selected point (latitude and longitude) will be fetched from the weather API.
    • The real-time weather conditions (such as temperature, humidity, and weather condition) will be displayed in a popup when you click on the location marker.
  • Search Locations:
    • A search box is available at the bottom of the map for easy location search.
    • You can enter any location (city name, landmark, etc.) to find it on the map.
    • Press Enter after typing a location, and the map will zoom to that location.
    • The application will place a marker at the searched location and show the weather conditions for that point.
